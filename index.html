Flight Management System

ENSF 337

The Flight Management System is a C++ program for managing airline flight information. It offers features like displaying seat maps, managing passenger data, and saving information to files. Users interact with the system through a menu-based interface, allowing them to add or remove passengers and view flight details. The program demonstrates object-oriented programming principles and file handling in C++, making it suitable for small-scale airline management or as an educational tool for learning C++ concepts.
